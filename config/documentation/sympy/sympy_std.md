SymPy is a Python library for symbolic mathematics.

Useful APIs include `sympy.symbols`, `sympy.Integer`, `sympy.Rational`, `sympy.I`, `sympy.pi`, expression construction with arithmetic operators, `sympy.sin`, `sympy.cos`, `sympy.exp`, `sympy.log`, `sympy.sqrt`, `sympy.expand`, `sympy.factor`, `sympy.simplify`, `sympy.cancel`, `sympy.together`, `sympy.diff`, `sympy.integrate`, `sympy.limit`, `sympy.solve`, `sympy.Eq`, `sympy.Matrix`, and polynomial helpers.

Generate valid Python programs that create symbolic expressions, equations, matrices, polynomials, sums, products, derivatives, integrals, limits, or simplification pipelines. Keep generated programs self-contained and deterministic.
