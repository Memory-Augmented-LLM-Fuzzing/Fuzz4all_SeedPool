Cirq is a Python framework for creating, editing, transforming, and simulating quantum circuits.

Useful APIs include `cirq.LineQubit`, `cirq.GridQubit`, `cirq.Circuit`, common gates such as `cirq.X`, `cirq.Y`, `cirq.Z`, `cirq.H`, `cirq.CNOT`, `cirq.CZ`, `cirq.rx`, `cirq.ry`, and `cirq.rz`, and circuit utilities such as `cirq.measure`, `cirq.inverse`, `cirq.resolve_parameters`, `cirq.Simulator`, `cirq.optimize_for_target_gateset`, and `cirq.QasmOutput`.

Generate valid Python programs that build one or more Cirq circuits, compose moments and operations, use parameters, measurements, controlled operations, inverses, or simulation when appropriate. Keep generated programs self-contained and deterministic.
